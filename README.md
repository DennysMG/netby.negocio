# Capa de Negocio.

También se creó una librería para la capa de dominio, donde se definen todas las reglas de negocio, las entidades principales y las validaciones necesarios para el comportamiento del sistema. Esta capa es independiente de la infraestructura y no contiene lógica de acceso a datos ni dependencias tecnológicas externas, estas se implementa el uso de DTOs como mecanismo transversal para el transporte de datos entre capas.
